## 🔧 常用命令

```bash
# 查看所有金融Agent
openclaw agents list | grep finance

# 启动总指挥
openclaw agent start finance_main

# 查看Agent状态
openclaw agent status finance_main

# 查看系统日志
openclaw logs finance_main --tail=50

# 停止Agent
openclaw agent stop finance_main
```
