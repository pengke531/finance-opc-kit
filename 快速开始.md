# 金融 OPC 系统快速开始

如果您只想用最短路径完成首次启用，请按下面做。

## 1. 安装

### Windows

```powershell
git clone https://github.com/pengke531/finance-opc-kit.git
cd finance-opc-kit
powershell -ExecutionPolicy Bypass -File .\install-finance-opc.ps1
```

### macOS / Linux

```bash
git clone https://github.com/pengke531/finance-opc-kit.git
cd finance-opc-kit
chmod +x install.sh
./install.sh
```

## 2. 验证 Agent 已注册

```bash
openclaw agents list
openclaw config validate
```

您应该能看到：

- `finance_main`
- `finance_data`
- `finance_analysis`
- `finance_trading`
- `finance_monitor`

## 3. 重启并新建会话

安装完成后：

1. 重启 OpenClaw 桌面端或 Gateway
2. 新建一个会话
3. 不要沿用旧会话

## 4. 先做两轮测试

第一句：

```text
@finance_main 你好
```

第二句：

```text
@finance_main 分析一下平安银行
```

如果第二句能正常返回分析结果，说明部署基本成功。

建议再补一轮：

```text
@finance_main 我想用模拟模式买入 000001.SZ，请给我一个分批建仓和止损方案
```

如果这句也能正常返回，说明主业务链路已经打通。

## 5. CLI 直连测试

如果您想直接用命令行测试：

```bash
openclaw agent --agent finance_main --message "分析一下平安银行"
```

## 6. 遇到问题先看这里

常见排查顺序：

1. `openclaw agents list`
2. `openclaw config validate`
3. `openclaw doctor`
4. 重启 OpenClaw
5. 新建会话后重新测试 `@finance_main`
