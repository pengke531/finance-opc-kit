# 📘 金融OPC系统 - 完整使用攻略

## 🎯 新手快速入门

### 第一次使用（5分钟上手）

#### 第1步：环境准备 (1分钟)
```bash
# 确认OpenClaw已安装
openclaw --version

# 确认有网络连接
ping api.eastmoney.com -c 3
```

#### 第2步：获取项目 (1分钟)
```bash
# 方式A：直接导入GitHub仓库（推荐）
openclaw import https://github.com/pengke531/finance-opc-kit

# 方式B：克隆后导入
git clone https://github.com/pengke531/finance-opc-kit.git
cd finance-opc-kit
openclaw import .
```

#### 第3步：验证安装 (1分钟)
```bash
# 检查Agent是否成功导入
openclaw agents list | grep finance

# 应该看到：
# finance_main     F01 Financial Commander
# finance_data     F02 Data Collector
# finance_analysis F03 Market Analyst
# finance_trading  F04 Trading Executor
# finance_monitor  F05 Market Monitor
```

#### 第4步：首次对话 (2分钟)
```bash
# 启动金融总指挥
openclaw chat finance_main

# 测试对话
你：你好，请介绍一下你的功能

F01：您好！我是F01 Financial Commander，金融OPC系统的总指挥。
我负责协调所有金融操作，包括数据收集、市场分析、交易执行和风险监控。
我可以帮您：
- 📊 分析股票技术面和基本面
- 💡 提供投资建议
- 🛡️ 管理投资风险
- 📈 制定投资策略
```

## ⚠️ 重要注意事项

### 风险提示
1. **投资风险**：所有投资建议仅供参考，不构成投资建议
2. **数据延迟**：金融数据可能有1-3秒延迟
3. **系统限制**：模拟交易，非真实交易环境
4. **责任自负**：所有投资决策由用户自己负责

### 使用限制
1. **不能代替人工判断**：系统建议需要结合个人判断
2. **不能预测黑天鹅**：无法预测突发事件
3. **不能保证收益**：过去表现不代表未来收益
4. **不能替代专业建议**：复杂情况需要咨询专业人士

## 🔧 故障排除指南

### 问题1：导入失败
```bash
# 症状：导入时报错
# 解决：检查OpenClaw版本
openclaw --version

# 症状：网络连接失败
# 解决：检查网络连接
ping api.eastmoney.com

# 症状：文件缺失
# 解决：重新克隆项目
git clone https://github.com/pengke531/finance-opc-kit.git
```

### 问题2：Agent无响应
```bash
# 症状：对话时Agent无响应
# 解决：重启Agent
openclaw agent restart finance_main

# 症状：Agent报错
# 解决：查看错误日志
openclaw logs finance_main --tail=50

# 症状：Agent启动失败
# 解决：检查Agent配置
openclaw agent config finance_main
```

### 问题3：数据获取失败
```bash
# 症状：无法获取股票数据
# 解决：测试网络连接
curl https://api.eastmoney.com

# 症状：数据异常
# 解决：清除缓存
openclaw memory clear

# 症状：技能加载失败
# 解决：重新导入
openclaw reload
```
